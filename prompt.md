# Prompt

A website front end similar to https://v2.flightclub.dance/login and based on the code from https://github.com/AdrianPotter/flight-club-frontend-v2.

The website should have:

- a front page with a logo and a short description, followed by login/register buttons
- an about page
- a login page taking email and password
- a register page that requires email, password, confirm password, and requires clicking a box confirming to terms and conditions
- a move list page that shows a list of acrobatics moves along with a search box that filters based on move names

Each page should have a footer containing simple information, and a navigation bar at the top.

Use the colour pallette from: https://coolors.co/ffffff-00171f-003459-007ea7-00a8e8

There should be a prompt.md with a copy of this prompt along with a readme.md containing instructions on how to run an deploy.

The website should use the Vue.js javascript framework.

It should be deployable as a single page application to an S3 bucket. Infrastructure as code will be added later.
